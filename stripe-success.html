Thanks for subscribing to OTTDNS, you may need to update your DNS settings on your devices or router to ensure you're using our DNS servers. Here's how:

Windows 10

  1. Click on the Start menu and open Settings.
  2. Click on Network & Internet.
  3. Select the network connection you want to change and click Change adapter options.
  4. Right-click on the network connection and select Properties.
  5. Scroll down and select Internet Protocol Version 4 (TCP/IPv4).
  6. Click on Properties.
  7. Select Use the following DNS server addresses.
  8. Address and enter the new DNS server addresses.
  9. Click OK to save the changes.

MacOS

  1. Click on the Apple menu and open System Preferences.
  2. Click on Network.
  3. Select the network connection you want to change and click Advanced.
  4. Click on the DNS tab.
  5. Click on the + button and enter the new DNS server addresses.
  6. Click on the - button to remove any existing DNS servers.
  7. Click OK to save the changes.

iOS

  1. Open the Settings app.
  2. Tap on Wi-Fi.
  3. Find the network you want to change and tap the â“˜ icon next to it.
  4. Tap on Configure DNS.
  5. Select Manual and tap Add Server.
  6. Enter the new DNS server addresses.
  7. Tap Save to save the changes.

Android

  1. Open the Settings app.
  2. Tap on Wi-Fi.
  3. Find the network you want to change and tap and hold on it.
  4. Select Modify network.
  5. Tap on Show advanced options.
  6. Under IP settings, select Static.
  7. Under DNS 1 and DNS 2, enter the new DNS server addresses.
  8. Tap Save to save the changes.

Router

  1. Open your web browser and type your router's IP address in the address bar.
  2. Enter your router's username and password to log in.
  3. Find the DNS settings page.
  4. Enter the new DNS server addresses and save the changes.

Note: It's important to note that updating DNS settings on a router will affect all devices connected to that network, so make sure to inform everyone who uses the network about the changes.

For more information on changing DNS settings, check out our FAQ page.
